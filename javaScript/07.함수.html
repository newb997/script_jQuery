* 선언적 함수(function 하고 이름쓰는거)

        function 함수명(매개변수 있으면 쓰고 없으면 안쓰고 필요한만큼[매개변수, 매개변수,...]) {
            함수호출시 실행할 코드;
            return 결과값;  있으면 넣고 없으면 생략
        }

* 익명 함수(이름없는함수

        변수 | 속성 = function(매개변수 있으면 넣고 없으면 안넣고 필요한만큼[매개변수, 매개변수,...]) {
            실행할 코드;
            return 결과값;  있으면 넣고 없으면 생략
        }
- 특정 변수나 속성에 대입되는 함수 제시시 주로 사용됨 (주로 이벤트 핸들러 작성시 사용)

* 화살표 함수(Arrow Function

        - 본문에 리턴만 있을 때
          변수 = ([매개변수, 매개변수,...]) => 리턴값;
        - 본문에 내용이 여러줄 일때
          변수 = ([매개변수, 매개변수,...]) => {
            실행코드;
            실행코드;
            ...
            return 결과값;  있으면 넣고 없으면 생략
          }
        - 익명함수를 간단히 작성하는 방법(ES6(ES2015)에서 추가)

* 즉시 실행 함수(iife(Immediately Invoked Function Expression))

        - 정의하면서 즉시 실행되는 함수
        - 단 한번만 호출됨
          ([변수] = function([매개변수, 매개변수,...]) {
            실행될 코드;
          })(인자,...);  전달인자
