            해당 요소에 해당 이벤트 발생시 실행할 내용;
        })

* 방법2. on 메소드를 이용한 방법

        $('선택자').on('이벤트명',function() {
            해당 요소에 해당 이벤트 발생시 실행할 내용;
        });

마우스 클릭 및 올려보세요

            선택된 상위요소 안에 존재하는 하위요소에 해당 이벤트가 발생했을 때 실행할 내용;

* 키보드 관련 이벤트

- keydown : 키누름 | keypress : 키가 눌려질때 발생하는 이벤트
-> keydown : 키보드의 모든키가 눌려질 때 발생됨
-> keypress : 키보드의 펑션키(f1, f2...), 기능키(alt,shift...), 한글 제외한 나머지 키가 눌려질 때 발생됨
- keyup : 키가 눌렸다 떼어질 때 발생하는 이벤트. keydown이 발생해야 keyup이 발생함.
