Content(innerHTML, innerText) 관련 메소드

* html(['문구']) : innerHTML속성과 관련된 메소드

선택된 요소의 content영역의 값을 리턴해주거나 또는 변경해주는 메소드
-> $('선택자').html() : 선택된 요소의 content영역의 값을 반환(html태그까지 포함)
-> $('선택자').html('문구') : 선택된 요소의 content 영역값을 해당 문구로 변경(html 태그가 존재하면 태그로 해석)

네이버로

* text(['문구']) 메소드 : innerText 속성과 관련된 메소드

선택된 요소의 content 영역 값을 리턴해주거나 또는 변경해주는 메소드
-> $('선택자').text() : 선택된 요소의 content영역의 텍스트를 반환
-> $('선택자').text('문구') : 선택된 요소의 content영역의 값을 '문구'로 변경(태그도 텍스트로 출력)
